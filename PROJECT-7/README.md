# Анализ динамики роста ВВП Ганы (1960–2021)

Этот проект посвящён визуализации и исследованию экономического роста Ганы на основе временного ряда данных ВВП за период с 1960 по 2021 год.

---

## Описание

В данной работе проведён углублённый анализ динамики ВВП Ганы с использованием различных методов временных рядов, включая:

- Построение коррелограмм для оценки стационарности и порядка дифференцирования ряда.
- ARIMA-моделирование с учётом выявленных параметров.
- Прогнозирование и визуализация результатов с интервалами доверия.
- Декомпозиция временного ряда на тренд, сезонность и остатки.
- Исследование волатильности ВВП и её квадратов с помощью скользящих средних.
- Сравнение фактических данных и прогнозов по модели GARH и линейной регрессии.
- Использование модели Prophet для прогнозирования ВВП.
- Визуализация анализа с акцентом на корректное отображение временной оси.

---

## Особенности визуализаций

- Для устранения проблем с отображением дат в Plotly даты преобразованы в строковый формат 'YYYY-MM-DD', обеспечивая чёткую и читабельную временную шкалу.
- Используются комплексные графики с несколькими субплотами для одновременного анализа трендов, сезонной компоненты и остатков.
- Выделены визуальные элементы, отражающие фактические данные, прогнозы и доверительные интервалы.
- Представлены сравнительные графики полной модели и прогноза вне выборки для контроля качества предсказаний.
- Применено скользящее среднее для сглаживания показателей волатильности.

## Как использовать

1. Загрузить и подготовить данные ВВП Ганы с 1960 по 2021 гг. в формате временного ряда.
2. Запустить ноутбук с выполнением всех блоков для генерации визуализаций и проведения анализа.
3. Изучить результаты визуализаций и интерпретировать экономические выводы.
4. При необходимости скорректировать параметры моделей для улучшения прогноза.

---

## Итог

Данный проект помогает понять структуру экономического роста Ганы, выделить ключевые тренды и сезонные влияния, а также оценить качество прогнозов различных моделей временных рядов с использованием интерактивных визуализаций.
