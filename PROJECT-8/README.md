# A/B-тестирование: Анализ конверсии и среднего чека

## Описание проекта

Данный проект представляет собой статистический анализ результатов A/B-теста, направленного на оценку двух версий дизайна посадочной страницы. Основные метрики для оценки — конверсия и средний чек покупателей.

Цель проекта — определить, какая из версий страницы более эффективна с точки зрения привлечения покупателей и увеличения выручки.

## Структура анализа

1. Формулировка гипотез  
   - Проверка равенства конверсий в группах A и B  
   - Проверка равенства средних чеков в группах A и B

2. Проверка нормальности данных  
   - Использование теста Шапиро-Уилка для среднего чека  
   - Выбор метода статистического тестирования в зависимости от нормальности

3. Проведение статистических тестов  
   - Z-тест для разности конверсий  
   - t-тест или тест Манна-Уитни для среднего чека

4. Построение доверительных интервалов (95%)  
   - Для конверсий в каждой группе и их разницы  
   - Для ежедневного среднего чека в каждой группе

5. Интерпретация результатов и выводы

## Используемые инструменты

- Python  
- Библиотеки: pandas, numpy, scipy, statsmodels

## Как запустить

1. Клонируйте репозиторий  
2. Установите необходимые библиотеки:
   
   pip install pandas numpy scipy statsmodels
   
3. Подготовьте данные по посещениям, покупкам и средним чекам в формате pandas DataFrame  
4. Запустите скрипт анализа

## Результаты

После выполнения кода вы получите:

- Результаты теста Шапиро-Уилка для проверки нормальности  
- p-value и статистики z-теста для конверсий  
- p-value и статистики t-теста или теста Манна-Уитни для среднего чека  
- 95% доверительные интервалы для конверсий и среднего чека  
- Рекомендации по выбору более эффективного варианта посадочной страницы
